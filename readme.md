# Hook

> build and tested for macOS (might work on linux as well)

## Install

make the tool executable

```
chmod +x .folder-hooker
```

add the following to your `.bash_profile`

```
source [PATH-WHERE-HOOK-IS-LIVING].folder-hooker
```
